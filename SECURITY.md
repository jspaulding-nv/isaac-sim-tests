# Security

Do not report product security vulnerabilities by committing sensitive logs or
opening an issue containing credentials, private network details, proprietary
assets, or personal information.

This repository's WebRTC viewer configuration does not provide authentication
or TLS termination. Run it only on a trusted private network or place it behind
appropriate access controls. Never expose the default deployment directly to
the public Internet.

For test-output privacy guidance, see [docs/privacy.md](docs/privacy.md).
